## :open_book: How to run
1. Clone this repository

git clone https://github.com/renijf/Tugas6_CMS.git

2. Install dependencies

composer install

3. Migrate the tables

php artisan migrate
php artisan db:seed --class=UserSeeder

5. Start the server

php artisan serve

6. Login

http://127.0.0.1:8000/login
